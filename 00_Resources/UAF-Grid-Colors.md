# UAF GRID COLORS
The UAF Grid uses a specific color palette to ensure visual consistency and accessibility across domains (i.e., the rows in the grid).

The Unified Architecture Framework (UAF) Traceability (Informative) document is Appendix A in the Version 1.2 UAF Specification.

* Link to [UAF 1.2 Specification](https://www.omg.org/spec/UAF/1.2)

* Link to [UAF Traceability (Informative) Document - formal/22-07-07](https://www.omg.org/cgi-bin/doc?formal/22-07-07.pdf)
